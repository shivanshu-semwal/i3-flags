# i3

`i3` is a `tiling window manager` designed for `X11`, inspired by `wmii` and written in `C`.

## Popular forks of i3

- `i3wm`
- `i3-flags`

### Mode in which windows can be arranged

- Tiling mode (default)
- Stacking
- Tabbed
- Floating

### Some default programs with i3

- `dmenu`
- `i3-bar`
- `i3-lock`

## Configuration

- `i3` - `~/.conf/i3/conf`
- `i3-status` - `~/.conf/i3-status/conf`
